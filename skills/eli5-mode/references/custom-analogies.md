# Custom Analogy Bank

Add your own analogies here. Claude will check this file alongside the built-in
analogy bank whenever eli5-mode is active.

Format: one concept per block, following the template below.

---

## How to add an analogy

```
**[Concept]**
[Your analogy — written as if explaining to the audience for your preferred level.]
```

Example:
```
**GraphQL**
Instead of ordering from a fixed menu (REST), GraphQL is like telling the chef
exactly what ingredients you want and getting back only those — nothing more.
```
